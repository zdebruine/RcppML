# Convert assessment results to a one-row data frame

Convert assessment results to a one-row data frame

## Usage

``` r
# S3 method for class 'nmf_assessment'
as.data.frame(x, ...)
```

## Arguments

- x:

  An `nmf_assessment` object.

- ...:

  Ignored.

## Value

A one-row data frame with all metrics as columns.
