# Read Pre-Stored Transpose from a StreamPress File

Read Pre-Stored Transpose from a StreamPress File

## Usage

``` r
st_read_transpose(path)
```

## Arguments

- path:

  Path to a `.spz` file.

## Value

A `dgCMatrix` representing CSC(A^T).
