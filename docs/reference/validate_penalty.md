# Validate and expand penalty vectors

Validate and expand penalty vectors

## Usage

``` r
validate_penalty(penalty, name, allow_negative = FALSE)
```

## Arguments

- penalty:

  Numeric scalar or length-2 vector

- name:

  Character name of penalty for error messages

- allow_negative:

  Logical; allow negative values (default FALSE)

## Value

Length-2 numeric vector `c(w, h)`
